WP Module

Defined in: wp.js:35

A WP REST API client for Node.js

Example:

var wp = new WP({ endpoint: 'http://src.wordpress-develop.dev/wp-json' });
wp.posts().then(function( posts ) {
    console.log( posts );
}).catch(function( err ) {
    console.error( err );
});

This module provides the following classes:

  • CollectionRequest
  • MediaRequest
  • PagesRequest
  • PostsRequest
  • TaxonomiesRequest
  • TypesRequest
  • UsersRequest
  • WP
  • WPRequest

This module is a rollup of the following modules:

  • CollectionRequest
    CollectionRequest extends WPRequest with properties & methods for filtering collections via query parameters. It is the base constructor for most top-level WP instance methods.
  • MediaRequest
    MediaRequest extends CollectionRequest to handle the /media API endpoint
  • PagesRequest
    PagesRequest extends CollectionRequest to handle the /posts API endpoint
  • PostsRequest
    PostsRequest extends CollectionRequest to handle the /posts API endpoint
  • TaxonomiesRequest
    TaxonomiesRequest extends CollectionRequest to handle the /taxonomies API endpoint
  • TypesRequest
    TypesRequest extends CollectionRequest to handle the /taxonomies API endpoint
  • UsersRequest
    UsersRequest extends CollectionRequest to handle the /users API endpoint. The /users endpoint responds with a 401 error without authentication, so users() forces basic auth.
  • WPRequest
    WPRequest is the base API request object constructor
